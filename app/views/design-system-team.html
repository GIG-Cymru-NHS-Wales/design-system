Design system team

The user-centred design (UCD) team at Digital Health and Care Wales maintains the design system. We work with colleagues across NHS Wales to research and develop components, patterns and templates based on evidence.

  • Rachel Meese - Head of UCD
  • Ramon Delgado - Lead Interaction Designer
  • Emma Lynham - Senior Content Designer

Email us if you have any questions, including if you'd like to request a new design asset.

How we manage new design asset requests

1. Request

The UCD team receives a request for a new design asset. We add it to our backlog.

2. Evaluation

The UCD team meets weekly to:

  • review the backlog
  • decide which requests we can accept
  • prioritise accepted requests by how much value they will add and how complex they are

We then contact the requester to let them know if we're able to create the asset.

3. Designing and testing

Once a new design asset is agreed and prioritised, a team is assigned to:

  • create a working prototype
  • run usability and accessibility testing sessions with users
  • develop the design asset from the tested prototype

4. Publishing

We publish the design asset on the website with a description of how and when to use it.

We also publish our documentation and version control in our Design System repository on GitHub. Documentation for project work will be published on the DHCW SharePoint site.
